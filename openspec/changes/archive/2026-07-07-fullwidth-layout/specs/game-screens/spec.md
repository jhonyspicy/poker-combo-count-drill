## ADDED Requirements

### Requirement: フルワイドレイアウト
全4画面（トップ・プレイ・リザルト・遊び方）はデバイスの画面幅いっぱいに表示する SHALL。横方向の余白制限（最大幅・外側パディング）を設けない。

#### Scenario: スマホでのトップ画面表示
- **WHEN** スマホでトップ画面を表示する
- **THEN** コンテンツがデバイス幅いっぱいに広がり、左右に余白が生じない

#### Scenario: スマホでのプレイ画面表示
- **WHEN** スマホでプレイ画面を表示する
- **THEN** 問題文・4択ボタンがデバイス幅いっぱいに広がる
