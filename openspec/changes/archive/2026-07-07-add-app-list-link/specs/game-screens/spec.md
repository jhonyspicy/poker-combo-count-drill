## MODIFIED Requirements

### Requirement: トップ画面
タイトルブロック・自己ベストカード（または初回案内文）・スタートボタン・遊び方へのリンク・アプリ一覧へのリンクを縦一列に表示する SHALL。

#### Scenario: 自己ベストがない場合
- **WHEN** localStorage に自己ベストが存在しない状態でトップ画面を表示する
- **THEN** 自己ベストカードの代わりに初回訪問向けの案内文を表示する

#### Scenario: 自己ベストがある場合
- **WHEN** localStorage に自己ベストが存在する状態でトップ画面を表示する
- **THEN** 最高連続正解数・到達レベル・達成日を表示するカードを表示する

#### Scenario: スタートボタンタップ
- **WHEN** ユーザーがスタートボタンをタップする
- **THEN** プレイ画面に遷移する

#### Scenario: アプリ一覧リンクタップ
- **WHEN** ユーザーが「アプリ一覧」リンクをタップする
- **THEN** `https://jhonyspicy.github.io/` が新しいタブで開く
