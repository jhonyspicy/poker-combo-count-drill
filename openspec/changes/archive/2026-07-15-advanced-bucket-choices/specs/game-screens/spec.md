# game-screens delta: 4択ボタンのラベル表示

## ADDED Requirements

### Requirement: 4択ボタンのラベル表示
4択ボタンは問題データが持つ選択肢ラベルを表示する SHALL。初級・中級では数値、上級ではコンボ数帯（例:「11〜40」）がラベルになる。正誤判定は選択肢の正解フラグで行う SHALL（表示ラベルと answer 数値の比較に依存しない）。上級の4択ボタンは常に昇順の固定順で表示する SHALL。

#### Scenario: 上級のバケットボタン表示
- **WHEN** 上級問題が表示される
- **THEN** 4択ボタンには「0〜10」「11〜40」「41〜80」「81以上」が昇順で表示され、問題が変わっても並び順は変わらない

#### Scenario: 上級の正解タップ
- **WHEN** 上級問題で負けコンボ数が属する帯のボタンをタップする
- **THEN** 正解として扱われ、連続正解数が +1 される

#### Scenario: 上級の誤答タップ
- **WHEN** 上級問題で負けコンボ数が属さない帯のボタンをタップする
- **THEN** 正解の帯が緑・選んだ帯が赤でハイライトされ、ゲームオーバーとなる

#### Scenario: 初級・中級の数値ボタン表示
- **WHEN** 初級または中級の問題が表示される
- **THEN** 4択ボタンには従来通り数値（シャッフル済み）が表示される

#### Scenario: リザルト画面の正解表示
- **WHEN** 上級問題でゲームオーバーになりリザルト画面が表示される
- **THEN** 正解として正確な負けコンボ数（帯ではなく数値）と解説が表示される
