## Purpose
ポーカーの役評価を担う。5〜7枚のカードから最強の5枚役を数値スコアとして評価し、2ハンドの勝敗（勝ち/負け/引き分け）を比較できるようにする。上級問題の正解計算の根幹。

## Requirements

### Requirement: 5枚役の評価
5枚のカードからポーカーの役を評価し、強さを単一の数値スコアとして返す SHALL。役カテゴリはハイカード < ワンペア < ツーペア < スリーカード < ストレート < フラッシュ < フルハウス < フォーカード < ストレートフラッシュの順で、同カテゴリ内はキッカーを含むタイブレークで比較できる SHALL。A-2-3-4-5（ホイール）は 5 ハイのストレートとして扱う SHALL。

#### Scenario: 役カテゴリの大小
- **WHEN** フラッシュの5枚とストレートの5枚を評価する
- **THEN** フラッシュのスコアがストレートのスコアより大きい

#### Scenario: 同カテゴリのタイブレーク
- **WHEN** ワンペア（A ペア）とワンペア（K ペア）を評価する
- **THEN** A ペアのスコアが大きい

#### Scenario: キッカー勝負
- **WHEN** 同じペアでキッカーだけが異なる2つのワンペアを評価する
- **THEN** キッカーの高い方のスコアが大きい

#### Scenario: ホイールストレート
- **WHEN** A-2-3-4-5 を評価する
- **THEN** 5 ハイのストレートとして評価され、6 ハイのストレートより弱い

### Requirement: 6〜7枚からの最強役の評価
6枚または7枚のカードから、5枚の組み合わせをすべて評価し最強のスコアを返す SHALL（ターン = ボード4枚+2枚、リバー = ボード5枚+2枚に対応）。

#### Scenario: 7枚からの最強役
- **WHEN** ボード5枚とハンド2枚の計7枚を評価する
- **THEN** C(7,5) = 21 通りのうち最強の5枚役のスコアが返る

#### Scenario: ボードプレイ
- **WHEN** ボード5枚がそのまま最強の役で、ハンド2枚が役を改善しない
- **THEN** ボード5枚の役のスコアが返り、同じボードの2ハンドは同スコア（チョップ）になる

### Requirement: 勝敗の比較
2つのハンド（同一ボード上）の勝敗はスコアの数値比較で判定する SHALL。スコアが等しい場合はチョップ（引き分け）であり、勝ちにも負けにも数えない SHALL。

#### Scenario: 勝敗判定
- **WHEN** 自分のスコアと相手コンボのスコアを比較する
- **THEN** 相手のスコアが大きければ「負けているコンボ」、小さければ「勝っているコンボ」、等しければどちらにも数えない

### Requirement: 評価ロジックのテスト
ハンド評価は正解計算の根幹であるため、Vitest による自動テストを備える SHALL。テストは少なくとも全役カテゴリの順序・タイブレーク・ホイール・ボードプレイ（チョップ）を検証する SHALL。

#### Scenario: テストの実行
- **WHEN** テストスイートを実行する
- **THEN** ハンド評価のテストがすべて成功する
